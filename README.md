# Data Filtering Tool using Pandas

This project is a simple Data Filtering Tool built using Python and Pandas.  
It demonstrates how to filter data using multiple conditions instead of traditional if-else statements.
The program:
- Creates a dataset
- Applies Boolean filtering
- Displays filtered results
- Combines multiple conditions
- Saves filtered data into a new CSV file
